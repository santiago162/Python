## Cómo ejecutar

```bash
python index.py
```

Desde el menú, selecciona el número del algoritmo que deseas ejecutar. Escribe `0` para salir.

> **Requisito:** tener instalada la librería `colorama`.
> ```bash
> pip install colorama
> ```
